# Trakines

Trakines shows the **layout** view of a level on your screen (solid background, flat silhouettes) while streaming the **full decorated** level to OBS through a **Spout2** sender.

- On-screen: layout mode (like XDBot's layout mode) applied only to the play layer.
- OBS (Spout2 sender `Trakines`): the original, fully decorated frame.
- Menus, pause, and the editor are mirrored 1:1 (no layout mode there by default).

Spout2 SDK is bundled inside the mod, so nothing extra needs to be installed.
